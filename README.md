Introduction:
The Exhibition Booth App is a cross-platform mobile application built with Flutter and powered by Firebase as its backend. It provides a complete digital ecosystem for managing exhibitions,  from booth creation and floor plan design to exhibitor registration, application approval, and payment processing. The app supports four distinct user roles: Guest, Exhibitor, Organizer, and Admin, each with a tailored experience and role-specific dashboards.
